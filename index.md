# Dashboard

Records of YummyOrder development progress.

## About Project

该项目的主题是“外卖点餐”,即设计一个云+端移动应用框架，来实现客户端点餐与web端商家交互的功能。同时，在实现外卖点餐的基本功能基础上，进行创新性的拓展，使产品更具吸引力，竞争性。
